Git Setup

# Install Git https://git-scm.com/downloads # Install VS Code https://code.visualstudio.com/ # Check Git version git --version # Configure Git git config --global user.name "Your Name" git config --global user.email "email@example.com"

Creating and Cloning Repository

# Create new repo on GitHub.com # Clone repository in VS Code # (Command Palette: Git: Clone) git clone https://github.com/YOUR-USERNAME/REPO-NAME.git # Navigate to cloned repository cd REPO-NAME

Git Basics

# Check status of your repository git status # Add files to staging area git add . # Commit changes git commit -m "Your message" # Pull changes from remote git pull origin main # Push changes to remote git push origin main

Branching

# Create new branch git branch feature-name # Switch to branch git checkout feature-name # Create and switch (shortcut) git checkout -b new-feature # Merge branch to main git checkout main git merge feature-name # Delete branch after merging git branch -d feature-name

Working with Remote Repositories

# Add a remote repository git remote add origin https://github.com/USERNAME/REPO.git # View remote repositories git remote -v # Fetch changes from remote git fetch origin # Pull changes from remote git pull origin main # Push changes to remote git push origin main # Remove a remote git remote remove origin

Pull Requests and Code Reviews

# Create a new branch for your feature git checkout -b new-feature # Make changes and commit git add . git commit -m "Add new feature" # Push the branch to GitHub git push origin new-feature # On GitHub: # 1. Go to your repository # 2. Click "Pull requests" # 3. Click "New pull request" # 4. Select your branch # 5. Click "Create pull request" # 6. Add description and create # Reviewing in VS Code: # 1. Install GitHub Pull Requests extension # 2. View pull request in VS Code # 3. Add comments and suggestions # 4. Approve or request changes # Merge pull request on GitHub
